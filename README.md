# 💳 Credit Card Fraud Detection

## 🚀 Overview
Machine Learning project to detect fraudulent transactions using classification models.

## 📊 Dataset
- Source: Kaggle Credit Card Fraud Dataset
- Highly imbalanced dataset

## ⚙️ Tech Stack
- Python
- Scikit-learn
- Pandas, NumPy
- Streamlit

## 🧠 Models Used
- Logistic Regression
- Random Forest
- XG Boost

## 📈 Features
- Data preprocessing & scaling
- Model training & evaluation
- Interactive Streamlit UI

## 🖥️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app/app.py
